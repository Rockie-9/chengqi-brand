# 承器 · Chéng Qì

> A Vessel for the Long Arc · Brand Color System for Rockie Lin / NarraTect Advisory

## 關於

**承器**是一套合成三軸（對外定位 × 對內哲學 × 時間維度）的品牌色彩系統，承載十年職涯基礎與下一程軌跡。七色、三層時態、四種聲音，以 HSE × 觀察者 × 跨傳統求道者三個心理錨點為深層骨架。
